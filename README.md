# 🏥 Healthcare Expenditure Prediction using Machine Learning

## 📌 Overview

This project focuses on predicting Out-of-Pocket (OOP) healthcare expenditure risk in Bangladesh using machine learning techniques. The goal is to identify households at financial risk and support data-driven healthcare decision-making.

---

## 🎯 Objectives

* Predict healthcare expenditure risk
* Identify key influencing factors
* Compare multiple machine learning models

---

## 🚀 Features

* Data preprocessing and cleaning
* Multiple ML model implementation
* Model comparison and evaluation
* Basic web interface for prediction (prototype)

---

## 🛠️ Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib
* XGBoost
* LightGBM

---

## ⚙️ Models Used

* Decision Tree
* Random Forest
* Logistic Regression
* KNN
* SVM
* XGBoost
* LightGBM
* Voting Classifier

---

## 📊 Results

* Achieved ~73% accuracy using ensemble models
* LightGBM and XGBoost performed best
* Successfully classified high and low expenditure groups

---

## 📚 Dataset

* HIES 2022–23 (Bangladesh national survey data)

---

## 🌐 Web Interface

A prototype web interface was developed (currently runs on localhost).
